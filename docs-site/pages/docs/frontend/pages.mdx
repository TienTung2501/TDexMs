## Pools Page (`/pools`)

Lists all active liquidity pools with key metrics.

| Column | Description |
|--------|-------------|
| Pair | Token pair (e.g., ADA/TOKEN) |
| TVL | Total value locked in ADA |
| Volume 24h | Trading volume in last 24 hours |
| Fees 24h | Fee revenue in last 24 hours |
| APY | Estimated annual percentage yield |
| Actions | Trade / Add Liquidity buttons |

**Sorting:** Click column headers to sort by any metric.

## Pool Detail (`/pools/[id]`)

Detailed view of a specific pool with tabs:

1. **Overview** вҖ” TVL chart, volume chart, price chart, key stats
2. **Swap History** вҖ” Recent trades in this pool with direction, amounts, time
3. **Liquidity** вҖ” Add/remove liquidity forms
4. **Info** вҖ” Contract addresses, fee structure, pool parameters

## Orders Page (`/orders`)

Create and manage advanced orders.

### Order Types

| Type | Description | Trigger |
|------|-------------|---------|
| **Limit** | Buy/sell at a target price | Pool price reaches target |
| **DCA** | Invest fixed amounts at intervals | Time interval elapsed |
| **Stop-Loss** | Sell when price drops | Pool price falls below threshold |

## Portfolio Page (`/portfolio`)

Comprehensive wallet overview with four sections:

### Summary
- Total balance in USD and ADA
- Balance breakdown: Wallet / Locked in Orders / Locked in LP
- Token allocation pie chart
- Portfolio performance line chart

### Open Positions
- Active intents and orders with progress tracking
- Cancel and reclaim actions
- Expiration countdown timers

### Trade History
- Completed swaps with execution details
- Explorer links for each transaction
- Filter by status and type

### LP Positions
- Active liquidity positions across pools
- LP token value and share percentage
- One-click withdraw button

## Analytics Page (`/analytics`)

Protocol-wide metrics and charts:

| Metric | Description |
|--------|-------------|
| Total TVL | Sum of all pool reserves in USD |
| Volume 24h | Total trading volume in last 24 hours |
| Fees 24h | Total fee revenue in last 24 hours |
| Fill Rate | Percentage of intents successfully filled |
| Unique Traders | Number of distinct wallet addresses |

**Charts:**
- TVL over time (30 days)
- Volume bar chart (daily)
- Fee revenue trend

## Admin Page (`/admin`)

<Callout type="warning">
Admin page is only accessible to wallets matching the configured `ADMIN_ADDRESS`.
</Callout>

### Protocol Hub

Step-by-step protocol deployment:

1. **Deploy Settings** вҖ” Create settings UTxO with protocol parameters
2. **Deploy Factory** вҖ” Deploy pool factory contract
3. **Create Pool** вҖ” Bootstrap initial liquidity pool

### Revenue Dashboard

- Pending fees per pool
- One-click fee collection
- 30-day fee growth chart

### Solver Control

- Real-time solver status (running, batches, success rate)
- Manual solver trigger button
- Active intents and pending orders count

### Protocol Settings

- Update fee parameters
- Change admin address
- View on-chain state
