# Frontend

import { Cards, Card } from 'nextra/components'

The SolverNet DEX frontend is a modern single-page application built with Next.js 16 and React 19.

## Tech Stack

| Technology | Version | Purpose |
|-----------|---------|---------|
| **Next.js** | 16 | App Router, SSR, file-based routing |
| **React** | 19 | UI rendering with concurrent features |
| **Tailwind CSS** | 4 | Utility-first styling |
| **shadcn/ui** | latest | Pre-built accessible UI components |
| **Recharts** | 2 | Charts and data visualization |
| **TradingView** | Widget | Professional candlestick charts |
| **CIP-30** | вЂ” | Cardano wallet integration standard |

## Page Routes

| Route | Page | Description |
|-------|------|-------------|
| `/` | Landing | Hero section, feature overview |
| `/trade` | Trading | Swap card, price chart, recent trades, order book |
| `/pools` | Pool List | All liquidity pools with TVL, volume, APY |
| `/pools/[id]` | Pool Detail | Pool stats, chart, swap history, liquidity management |
| `/orders` | Orders | Create and manage Limit, DCA, Stop-Loss orders |
| `/portfolio` | Portfolio | Wallet overview, open positions, trade history, LP positions |
| `/analytics` | Analytics | Protocol stats, TVL chart, volume, fee revenue |
| `/admin` | Admin Panel | Protocol deployment, settings, fee collection, solver control |
