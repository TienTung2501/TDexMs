Production deployment guide for SolverNet DEX.

## Architecture

```
┌──────────────────┐     ┌──────────────────┐
│   Vercel          │     │   Render          │
│   (Frontend)      │────→│   (Backend API)   │
│   Next.js SSR     │     │   Express + WS    │
└──────────────────┘     └────────┬─────────┘
                                  │
                    ┌─────────────┼─────────────┐
                    │             │             │
              ┌─────▼─────┐ ┌────▼────┐ ┌─────▼──────┐
              │ Supabase   │ │Upstash  │ │ Blockfrost │
              │ PostgreSQL │ │ Redis   │ │ Cardano    │
              └────────────┘ └─────────┘ └────────────┘
```

## Quick Deploy Checklist

- [ ] PostgreSQL database provisioned (Supabase recommended)
- [ ] Blockfrost API key obtained (correct network)
- [ ] Redis cache set up (Upstash recommended)
- [ ] Backend deployed (Render / Docker)
- [ ] Frontend deployed (Vercel)
- [ ] Environment variables configured
- [ ] CORS origins set for production domain
- [ ] SSL/TLS enabled on all endpoints
- [ ] Database migrations applied
- [ ] Smart contracts deployed on-chain
- [ ] Solver wallet funded and enabled
- [ ] Monitoring and alerting set up
