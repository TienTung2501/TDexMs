# Quick Start

import { Callout, Steps, Tabs } from 'nextra/components'

Get SolverNet DEX running locally in under 5 minutes.

## Prerequisites

- **Node.js** ≥ 20.0.0
- **pnpm** 9.15.0+ (`corepack enable && corepack prepare pnpm@9.15.0 --activate`)
- **Docker** (optional, for local Cardano node)
- A **Blockfrost** API key (free at [blockfrost.io](https://blockfrost.io))

<Steps>
### Clone the repository

```bash
git clone https://github.com/TienTung2501/decentralize.git
cd decentralize
```

### Install dependencies

```bash
pnpm install
```

This installs dependencies for all workspace packages (backend, frontend, docs-site) using pnpm workspaces.

### Set up environment variables

```bash
cp backend/.env.example backend/.env
```

Edit `backend/.env` with your configuration:

```env
# Required
DATABASE_URL="postgresql://user:password@localhost:5432/solvernet"
BLOCKFROST_PROJECT_ID="preprodXXXXXXXXXXXXXXXXXXXXXXXXX"
CARDANO_NETWORK="preprod"

# Solver (optional — only if running a solver)
SOLVER_SEED_PHRASE="your twenty-four word mnemonic phrase ..."
SOLVER_ENABLED=false
```

See [Configuration](/docs/getting-started/configuration) for the full list of environment variables.

### Set up the database

```bash
cd backend
pnpm exec prisma migrate dev
pnpm exec prisma generate
```

### Start the backend

```bash
pnpm exec tsx src/index.ts
```

The API server starts on `http://localhost:3001`.

### Start the frontend

In a new terminal:

```bash
cd frontend
pnpm dev
```

The frontend starts on `http://localhost:3000`.

</Steps>
