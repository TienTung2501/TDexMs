# Quickstart

import { Callout, Steps, Tabs } from 'nextra/components'

Get the full SolverNet DEX stack running locally in 5 minutes.

<Callout type="warning">
**Prerequisites:** Node.js 18+, pnpm 8+, Git, and a Blockfrost API key (free at [blockfrost.io](https://blockfrost.io)).
</Callout>

<Steps>

### Clone & Install

```bash
git clone https://github.com/TienTung2501/decentralize.git
cd decentralize
pnpm install
```

### Set Up Database

<Tabs items={['Supabase (Recommended)', 'Docker', 'Local PostgreSQL']}>
  <Tabs.Tab>
    1. Create a free project at [supabase.com](https://supabase.com)
    2. Go to **Settings → Database → Connection string**
    3. Copy the connection string (use the "URI" format)
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash
    docker compose -f docker-compose.dev.yml up -d postgres
    # Connection: postgresql://solvernet:solvernet@localhost:5432/solvernet
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash
    createdb solvernet
    # Connection: postgresql://localhost:5432/solvernet
    ```
  </Tabs.Tab>
</Tabs>

### Configure Environment

```bash
cd backend
cp .env.example .env
```

Edit `.env` with your values:

```env
# Required
DATABASE_URL="postgresql://..."
BLOCKFROST_PROJECT_ID="preprodXXXXXXXXXXXXX"

# Network
CARDANO_NETWORK="preprod"

# Frontend connection
CORS_ORIGIN="http://localhost:3000"
```

### Initialize Database

```bash
cd backend
pnpm exec prisma db push
pnpm exec prisma generate
```

### Start Backend

```bash
cd backend
pnpm exec tsx src/index.ts
```

You should see:
```
[INFO] SolverNet DEX API starting...
[INFO] Database connected
[INFO] Server listening on http://0.0.0.0:3001
[INFO] WebSocket server ready on /v1/ws
```

### Start Frontend

In a new terminal:

```bash
cd frontend
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Verify

```bash
# Health check
curl http://localhost:3001/v1/health

# Should return:
# { "status": "healthy", "services": { "database": "connected", ... } }
```

</Steps>
