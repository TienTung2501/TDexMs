# API Reference

import { Cards, Card } from 'nextra/components'

The SolverNet DEX backend exposes a comprehensive REST API and real-time WebSocket interface.

## Base URL

```
Production: https://your-api-domain.com/v1
Local:      http://localhost:3001/v1
```

## Authentication

Currently, public endpoints do not require authentication. Admin endpoints verify the caller's wallet address matches the configured `ADMIN_ADDRESS`.

## Rate Limiting

| Type | Limit | Window |
|------|-------|--------|
| **Read** (GET) | `RATE_LIMIT_MAX` requests | 60 seconds |
| **Write** (POST/DELETE) | `RATE_LIMIT_MAX / 5` requests | 60 seconds |

When rate limited, the API returns HTTP 429 with:

```json
{
  "status": "error",
  "code": "RATE_LIMIT_EXCEEDED",
  "message": "Too many requests, please try again later"
}
```

## Response Format

All responses follow a consistent JSON structure:

```json
{
  "field": "value"
}
```

Error responses include:

```json
{
  "status": "error",
  "code": "ERROR_CODE",
  "message": "Human-readable description",
  "details": {}
}
```
