# REST API

import { Callout, Tabs } from 'nextra/components'

Complete reference for all REST endpoints. Base path: `/v1`

---

## Health

### `GET /health`

Check system health status.

**Response:**
```json
{
  "status": "healthy",
  "uptime": 3600,
  "database": "connected",
  "blockfrost": "connected",
  "cache": "connected",
  "chainTip": { "slot": 12345678, "hash": "abc..." }
}
```

### `GET /health/ready`

Kubernetes-style readiness probe. Returns `200` when all services are connected.

## Quote

### `GET /quote`

Get a swap quote with price impact calculation.

**Query Parameters:**

| Parameter | Type | Required | Description |
|---|---|---|---|
| `inputAsset` | string | Yes | Input asset unit (e.g., `lovelace`) |
| `outputAsset` | string | Yes | Output asset unit |
| `amount` | string | Yes | Input amount in smallest unit |
| `slippage` | number | No | Slippage tolerance (default: 0.5%) |

**Response:**
```json
{
  "success": true,
  "data": {
    "inputAmount": "100000000",
    "outputAmount": "49251234",
    "minimumOutput": "48996478",
    "priceImpact": "0.0050",
    "fee": "300000",
    "protocolFee": "50000",
    "effectivePrice": "0.4925",
    "route": {
      "type": "direct",
      "pools": ["pool_abc123"]
    }
  }
}
```

## Intents

### `POST /intents`

Create a new swap intent. Returns an unsigned transaction for the user to sign.

**Request Body:**
```json
{
  "creatorAddress": "addr_test1qz...",
  "inputAsset": "lovelace",
  "outputAsset": "abc123def456.TOKEN",
  "inputAmount": "100000000",
  "minimumOutput": "49000000",
  "expiresInSeconds": 3600
}
```

**Response:**
```json
{
  "success": true,
  "data": {
    "intentId": "intent_xyz789",
    "unsignedTx": "84a400...",
    "escrowAddress": "addr_test1wz...",
    "estimatedOutput": "49251234",
    "expiresAt": "2026-02-27T13:00:00Z"
  }
}
```

### `GET /intents`

List intents with optional filters.

**Query Parameters:**

| Parameter | Type | Description |
|---|---|---|
| `address` | string | Filter by creator address |
| `status` | string | Filter by status (ACTIVE, FILLED, etc.) |
| `limit` | number | Max results (default: 20) |
| `offset` | number | Pagination offset |

### `GET /intents/:id`

Get a specific intent by ID.

### `DELETE /intents/:id`

Cancel an active intent. Returns unsigned cancel transaction.

**Request Body:**
```json
{
  "creatorAddress": "addr_test1qz..."
}
```

**Response:**
```json
{
  "success": true,
  "data": {
    "intentId": "intent_xyz789",
    "unsignedTx": "84a400...",
    "status": "CANCELLING"
  }
}
```

## Pools

### `GET /pools`

List all liquidity pools.

**Query Parameters:**

| Parameter | Type | Description |
|---|---|---|
| `sort` | string | Sort by: `tvl`, `volume`, `apy` (default: `tvl`) |
| `order` | string | `asc` or `desc` (default: `desc`) |
| `limit` | number | Max results |

**Response:**
```json
{
  "success": true,
  "data": [
    {
      "id": "pool_abc123",
      "assetA": { "unit": "lovelace", "name": "ADA", "decimals": 6 },
      "assetB": { "unit": "abc...TOKEN", "name": "TOKEN", "decimals": 6 },
      "reserveA": "1000000000000",
      "reserveB": "500000000000",
      "totalLpTokens": "707106781",
      "fee": 30,
      "tvl": "2000000",
      "volume24h": "150000",
      "apy": "12.5",
      "state": "ACTIVE"
    }
  ]
}
```

### `GET /pools/:id`

Get detailed pool information.

### `POST /pools/create`

Create a new liquidity pool. Admin only.

**Request Body:**
```json
{
  "assetA": "lovelace",
  "assetB": "abc123.TOKEN",
  "amountA": "1000000000",
  "amountB": "500000000",
  "fee": 30,
  "creatorAddress": "addr_test1qz..."
}
```

### `POST /pools/:id/deposit`

Deposit liquidity into an existing pool.

**Request Body:**
```json
{
  "amountA": "100000000",
  "amountB": "50000000",
  "depositorAddress": "addr_test1qz...",
  "slippage": 0.5
}
```

### `POST /pools/:id/withdraw`

Withdraw liquidity from a pool.

**Request Body:**
```json
{
  "lpAmount": "70710678",
  "withdrawerAddress": "addr_test1qz...",
  "minA": "99000000",
  "minB": "49000000"
}
```

### `GET /pools/:id/history`

Pool history snapshots (hourly).

### `GET /pools/:id/swaps`

Recent swaps for a pool.

## Chart / OHLCV

### `GET /chart/config`

TradingView data feed configuration.

### `GET /chart/candles`

Get OHLCV candlestick data.

**Query Parameters:**

| Parameter | Type | Description |
|---|---|---|
| `poolId` | string | Pool identifier |
| `interval` | string | `H4`, `D1`, or `W1` |
| `from` | number | Start timestamp (Unix seconds) |
| `to` | number | End timestamp |
| `limit` | number | Max candles (default: 500) |

**Response:**
```json
{
  "success": true,
  "data": [
    {
      "timestamp": 1740650400,
      "open": "0.5012",
      "high": "0.5150",
      "low": "0.4980",
      "close": "0.5100",
      "volume": "15000000"
    }
  ]
}
```

### `GET /chart/price/:poolId`

Latest price for a pool.

### `GET /chart/history`

TradingView UDF-compatible history endpoint.

## Transaction

Submit a signed transaction to the Cardano network.

**Request Body:**

### `POST /tx/confirm`

Confirm a transaction and update related entities.

### `GET /tx/status/:txHash`

## Admin

Admin endpoints require the request to come from the configured `ADMIN_ADDRESS`. Authentication is verified via wallet signature.

Collect protocol fees from pools.
